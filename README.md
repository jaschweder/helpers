# Helpers
> Classes and functions for any project

## Install
```
composer require jaschweder\helpers
```

## Getting start
Examples:
```
// dump and die
dd('some random text');

// get enviorment with default value
env('FOO', 'bar');
```
> for more examples see the ´helpers.php´ file
